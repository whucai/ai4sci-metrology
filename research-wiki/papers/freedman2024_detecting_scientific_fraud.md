# Detecting Scientific Fraud Using Argument Mining

## One-line thesis
Retracted/fraudulent papers contain poorly formed and reasoned arguments compared to legitimate papers; argument mining + argument quality evaluation can detect them from abstracts with significant improvement over baselines.
